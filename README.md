# 🏦 API de Simulação Bancária

> **Status do Projeto:** Em Desenvolvimento 🚧

Uma API RESTful desenvolvida em **C#** e **.NET** para simular operações bancárias do mundo real. 

Este projeto foi criado com o objetivo de colocar em prática o desenvolvimento backend, focando fortemente na aplicação de **Programação Orientada a Objetos (POO)**, modelagem de domínio e validação de regras de negócio.

## 💻 Tecnologias e Ferramentas
- **Linguagem:** C#
- **Framework:** .NET (Web API)
- **Versionamento:** Git e GitHub
- **IDE:** Visual Studio Code

## 🧠 Conceitos Aplicados
Durante o desenvolvimento desta API, os seguintes conceitos de Engenharia de Software estão sendo aplicados e consolidados:
- **Programação Orientada a Objetos (POO):** Abstração, Encapsulamento, Herança e Polimorfismo.
- **Estruturas de Dados:** Manipulação de listas e coleções para armazenar o histórico de transações.
- **Tratamento de Exceções:** Prevenção de erros como saques com saldo insuficiente ou transferências inválidas.
- **Arquitetura Limpa:** Separação de responsabilidades (Controllers, Models, Services).

## ⚙️ Funcionalidades (Roadmap)
Abaixo estão as funcionalidades planejadas para o sistema:

- [ ] **Gerenciamento de Contas:** Criação de contas (Corrente e Poupança) com numeração única.
- [ ] **Operações Básicas:** Depósitos e Saques com validação de saldo e limites.
- [ ] **Transferências:** Envio de valores entre diferentes contas de forma segura.
- [ ] **Extrato Bancário:** Registro e listagem do histórico de todas as transações realizadas.
- [ ] **Regras de Negócio Avançadas:** Cobrança de taxas para contas correntes e rendimento de juros para contas poupança.
